在節奏快速、壓力沉重的都市生活中，人們渴望一個能夠放慢腳步、放鬆心靈的療癒空間。因此我們設計這間貓咪咖啡廳，希望透過溫柔可愛的貓咪陪伴與香醇咖啡，為顧客打造一處能暫時遠離喧囂、重新找回平靜與笑容的療癒角落。

本貓咖網頁以「韓系可愛 × 霧粉色」為主要視覺風格，搭配圓角版面設計、插畫風圖示與大量留白，營造出溫柔、夢幻又具質感的氛圍，讓使用者一進入網站就感受到輕鬆與幸福感。整體設計特別鎖定 IG 客群、情侶及網美族群，強調好拍、好分享的使用體驗，使網站本身成為吸引打卡與轉分享的重要入口。

我們期望透過療癒的貓咪、精緻的餐點與具有故事感的網站設計，打造一個讓人願意停留、分享並反覆造訪的溫暖療癒空間。
